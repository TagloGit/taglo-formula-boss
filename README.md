# Formula Boss

An Excel add-in that allows power users to write inline expressions using a concise DSL that transpiles to C# UDFs at runtime.

## Documentation

See [docs/specs and plan](docs/specs%20and%20plan/) for the full specification and implementation plan.

## Building

Requires .NET 10 SDK.

```bash
dotnet build formula-boss/formula-boss.slnx
```
